1.4. Используя Платформу, вы даёте согласие на обработку ваших персональных данных в соответствии с настоящей Политикой.

2. Какие данные мы собираем

2.1. Данные авторизации (автоматически из Telegram):

  • Telegram ID (числовой идентификатор);
  • имя и фамилия (если указаны в Telegram);
  • username (если указан в Telegram).

2.2. Данные, предоставляемые пользователем:

  • токены Telegram-ботов (хранятся в зашифрованном виде, AES-256);
  • токен Leadteh API (хранится в зашифрованном виде, AES-256);
  • идентификаторы ботов в системе Leadteh;
  • содержание рассылок (текст, фото).

2.3. Данные контактов рассылок:

  • Telegram ID контактов (получены из Leadteh API по поручению пользователя);
  • имя, username контактов (при наличии в Leadteh);
  • теги и принадлежность к спискам.

3. Цели обработки

  • авторизация и идентификация пользователей;
  • предоставление доступа к функциональности Платформы;
  • выполнение рассылок через Telegram Bot API;
  • учёт использования Платформы (лимиты тарифа);
  • техническая поддержка;
  • улучшение работы Платформы.

4. Правовые основания обработки

  • Согласие субъекта (ст. 6 ч.1 п.1 152-ФЗ) — при начале использования Платформы;
  • Исполнение договора (ст. 6 ч.1 п.5 152-ФЗ) — для выполнения условий Публичной оферты.

5. Хранение и защита данных

5.1. Данные хранятся на защищённом VPS-сервере на территории РФ.

5.2. Чувствительные данные (токены ботов, API-ключи) шифруются алгоритмом AES-256.

5.3. Доступ к серверу ограничен и защищён.

5.4. Данные хранятся в течение всего периода использования Платформы и 12 месяцев после прекращения использования, после чего удаляются.

6. Передача данных третьим лицам

6.1. Оператор не продаёт и не передаёт персональные данные третьим лицам, за исключением:

  • Telegram API — для выполнения рассылок (передаются: Telegram ID получателей, содержание сообщений). Обработка по поручению пользователя;
  • Leadteh API — для получения списка контактов и тегов. Обработка по поручению пользователя с использованием его токена;
  • по требованию закона — при наличии законного запроса уполномоченных органов.
